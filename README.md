# Web Dr. Montañés

## Link

[https://pacog.github.io/drmontanes/](https://pacog.github.io/drmontanes/)

## Run locally

Install [gulp](https://gulpjs.com/)

Install dependencies:
```bash
npm install
```
Type
```bash
gulp
```

To start a local server and watch the files so it autobuilds.

## Deploy to gh-pages

```bash
gulp deploy
```
